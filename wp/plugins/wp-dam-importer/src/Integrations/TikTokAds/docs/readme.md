-We recommend using **your own published OAuth app** to create a connection, as this means you will only have to refresh your access tokens once per year, instead of once per week which is mandated by TikTok for apps in development mode.<br><br>

However, if you do not want to go through the process of creating your own OAuth app, then all users have the option of using Medialake's default TikTok Business app instead.<br><br>

Medialake's OAuth app can be found when you go to create a connection in the Catalogues page, and then clicking on View Integration for TikTok.<br><br>

Please see our guide for creating your own TikTok OAuth app, if you wish to do so: <a href='https://medialake.atlassian.net/wiki/spaces/medialake/pages/xxxxxxx/TikTokAds' target='_blank'>How To Guide</a>.
