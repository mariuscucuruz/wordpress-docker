To configure Frontify integration, you will need the following credentials:
-
-
**OAuth 2.0 Authentication**
-
-
- Client ID
- Client Secret
- Tenant Name (e.g. `yourcompany` for `brand.yourcomapany.com`)
-
OR
-
-
**Developer API Key Authentication**
-
-
- Developer Token
- Tenant Name (e.g. `yourcompany` for `brand.yourcomapany.com`)
-
Ensure these values are set in your environment or configuration files before using the integration.
